# UpTimeRat

Simple UpTime monitoring application written in PHP, framework Symfony. Currently in development.

![UpTimeRat](https://i.imgur.com/EFCF2Mv.png)

# Features
* Working with HTTP, HTTPS and PORT
* Tailwind simple modern UI
* Public status pages
